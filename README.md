# Arpit Kushwaha Portfolio

Personal portfolio website for **Arpit Kushwaha**.

This site is built as a simple static website using:

- `HTML`
- `CSS`
- `JavaScript`

## Project Files

- `index.html` - portfolio structure and content
- `style.css` - design and responsive styling
- `script.js` - theme toggle and scroll animations
- `serve.js` - optional local development server

## Best Deployment Choice

For this project, the best option is **GitHub Pages** because:

- the site is fully static
- no database is required
- GitHub Pages is free and simple
- it works well with a portfolio hosted from a GitHub repository

## Run Locally

You can open `index.html` directly in the browser, or run the local server:

```powershell
node serve.js
```

Then open:

```text
http://localhost:5500
```

For phone testing on the same Wi-Fi, use your computer's local IP instead of `localhost`.

## Deploy With GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder to that repository.
3. Push the code to the `main` branch.
4. In GitHub, open:
   `Settings > Pages`
5. Under `Build and deployment`:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
6. Save the settings.
7. Wait a minute for GitHub Pages to publish the site.

Your site URL will usually look like:

```text
https://your-github-username.github.io/repository-name/
```
